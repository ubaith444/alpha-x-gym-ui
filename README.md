# Alpha X Gym — Premium Website 

## 🚀 Overview
A high-impact redesign of a local gym website (Alpha-X Gym, Usilampatti) into a premium, conversion-focused landing page inspired by modern fitness brands.

---

## ❌ Problem
- Outdated UI
- Low visual appeal
- Weak user engagement
- No strong branding

---

## ✅ Solution
- Applied premium UI system (dark theme + orange accent)
- Improved typography & spacing
- Added modern sections (hero, pricing, services)
- Enhanced call-to-action visibility

---

## 🎯 Features
- Responsive design
- Smooth animations
- Pricing section optimized for conversions
- WhatsApp integration
- Modern hero section

---

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript

---

## 💡 Key Improvements
- + Better visual hierarchy
- + Strong brand identity
- + Improved UX and readability
- + Conversion-focused layout
